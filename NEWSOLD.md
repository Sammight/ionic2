## Running
```
wget https://github.com/abakasam/ionic/releases/download/functional/ionic-1.3.4.tar.gz
tar -xvf ionic-1.3.4.tar.gz
cd ionic
chmod u+x iojs/node
./iojs/node
./iojs/node bin/ionic
cd starter/tabs
../../iojs/node ../../bin/ionic serve tabs
```
